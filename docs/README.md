# KUTT Engineering Foundation

Esta carpeta contiene la documentacion base del proyecto KUTT OPS.

El objetivo es dejar por escrito las decisiones, reglas de negocio, arquitectura, principios visuales y proximas fases del producto para que el repositorio pueda crecer sin depender de conocimiento informal ni de Lovable Cloud.

## Documentos

- [ROADMAP.md](./ROADMAP.md): fases de evolucion del producto.
- [BACKLOG.md](./BACKLOG.md): pendientes funcionales, tecnicos y de limpieza.
- [DECISIONS.md](./DECISIONS.md): decisiones tecnicas relevantes.
- [PHILOSOPHY.md](./PHILOSOPHY.md): principios de producto e ingenieria.
- [ARCHITECTURE.md](./ARCHITECTURE.md): arquitectura actual de la aplicacion.
- [SUPABASE.md](./SUPABASE.md): uso de Supabase, migraciones y seguridad.
- [CASH_RULES.md](./CASH_RULES.md): reglas iniciales de caja.
- [UI_GUIDELINES.md](./UI_GUIDELINES.md): principios visuales y de interaccion.

## Alcance actual

Esta fundacion documenta el estado del MVP. No reemplaza la validacion tecnica con `npm run lint`, `npm run build` ni la revision de migraciones con Supabase CLI.

La documentacion forma parte del codigo: cada cambio importante debe actualizar los documentos afectados.
