# Philosophy

KUTT existe para hacer mas simple la operacion diaria de una barberia.

## Principios

- La tecnologia debe reducir trabajo, no aumentarlo.
- Todo debe poder hacerse rapido desde movil.
- La caja nunca puede mentir.
- El diseno debe ser funcional.
- La documentacion forma parte del codigo.
- Cada feature debe resolver un problema real.

## Como tomar decisiones

Antes de agregar una funcion, preguntar:

- Que problema operacional resuelve?
- Reduce pasos o agrega carga?
- Funciona bien en movil?
- Mantiene la caja consistente?
- Puede explicarse y auditarse?

Si la respuesta no es clara, la funcion debe esperar.
